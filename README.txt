숙박업 영업지도 PWA 최종본

[GitHub Pages 업로드]
1. 이 폴더 안의 파일 6개를 저장소 최상단에 올립니다.
2. GitHub 저장소 상단 Settings → 왼쪽 Pages.
3. Build and deployment의 Source를 Deploy from a branch로 선택합니다.
4. Branch를 main, 폴더를 /(root)로 선택하고 Save합니다.
5. 약 1~3분 후 표시되는 https://계정명.github.io/저장소명/ 주소로 접속합니다.

[설치]
Android Chrome: 메뉴 → 앱 설치 또는 홈 화면에 추가
iPhone Safari: 공유 → 홈 화면에 추가
PC Chrome/Edge: 주소창 오른쪽 설치 아이콘

[중요]
- 방문상태, 메모, 직접 추가·수정한 업장은 현재 브라우저 기기에 저장됩니다.
- 휴대폰 교체나 브라우저 데이터 삭제 전에 반드시 '전체 백업'을 내려받으세요.
- 기본 핀은 권역별 임시 위치입니다. 업장 수정 → 지도에서 위치 선택으로 옮길 수 있습니다.
- 첫 실행과 지도 타일, 네이버지도 검색에는 인터넷이 필요합니다.
